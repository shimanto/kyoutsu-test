# 大学物語 — 共通テスト攻略プラットフォーム

東京大学・難関大合格を目指す受験生向けの学習プラットフォーム。  
**配点ヒートマップ** スタイルで全9科目900点を可視化し、忘却曲線に基づく最適な学習計画を自動生成します。

## 特徴

- **学習マップ** — 6教科ブロック×分野別小ブロックを配点比例で表示。正答率に応じて赤→緑にグラデーション変化
- **忘却曲線 (SM-2)** — 科学的アルゴリズムで最適な復習タイミングを自動計算
- **弱点自動検出** — 正答率の低い分野を特定し、重点ドリルを自動生成
- **タイムキーパー** — 本番までのロードマップと日次タスクを可視化
- **PWA対応** — スマホにインストールしてオフラインでも学習可能

| API | Hono on Cloudflare Workers |
| DB | Cloudflare D1 (SQLite) |
| Auth | LINE Login (LIFF SDK) |
| 共通パッケージ | `packages/shared` (型・定数・Zodスキーマ) |

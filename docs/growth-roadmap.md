# 大学物語 成長戦略ロードマップ

> 作成日: 2026-04-05
> ステータス: 策定完了・実行開始

---

## 1. ビジョン

**「すべての受験生に、自分だけの攻略マップを」**

共通テスト受験者（年間約50万人）のうち、まず国公立理系志望者（約10万人）をターゲットに、
配点ヒートマップ型の学習可視化と忘却曲線ベースの復習最適化で圧倒的な差別化を図る。

## 5. 競合分析・差別化

| 競合 | 特徴 | 大学物語の優位性 |
|------|------|----------------|
| スタディサプリ | 動画講義中心、月額¥2,178 | ヒートマップ可視化で弱点が一目瞭然 |
| atama+ | AI個別最適化、B2B | 個人利用可能、LINE連携で手軽 |
| 共通テスト過去問アプリ | 過去問のみ | AI変形問題で無限の演習量 |
| 赤本・青本 | 紙ベース | 忘却曲線で最適タイミング復習 |

**大学物語の3大差別化ポイント:**
1. **配点ヒートマップUI** — 全科目の習熟度を一画面で把握
2. **SM-2忘却曲線** — 科学的根拠に基づく復習最適化
3. **AI変形問題** — 過去問ベースの無限演習

## 7. KPIダッシュボード定義

アプリ内の `/admin/growth` ページで以下を追跡:

### North Star Metric
**週間アクティブ学習者数 (WAU)** — 週に1回以上ドリルを解いたユニークユーザー数

### Primary KPIs

| カテゴリ | 指標 | 計算方法 | 目標 |
|---------|------|---------|------|
| **Acquisition** | 新規登録数/週 | users テーブルの created_at | Phase別に設定 |
| **Activation** | 初回ドリル完了率 | 登録→初回ドリル完了 | > 60% |
| **Retention** | D7/D30継続率 | 7日後/30日後にドリル実施 | D7 > 40%, D30 > 20% |
| **Revenue** | MRR | 月額課金合計 | Phase別に設定 |
| **Referral** | 招待転換率 | 招待リンク経由の登録/招待送信 | > 15% |

### Secondary KPIs

| 指標 | 計算方法 |
|------|---------|
| 平均セッション時間 | study_sessions の duration 平均 |
| 1人あたり問題数/日 | answers テーブルの日次集計 |
| NPS | フィードバックのスコア集計 |
| AI変形問題利用率 | AI問題回答数 / 全回答数 |
| ヒートマップ閲覧回数 | ダッシュボードページビュー |

## 8. リスクと対策

| リスク | 影響度 | 対策 |
|--------|-------|------|
| コンテンツ不足 | 高 | AI変形問題で問題数を倍増、OCR パイプラインの自動化 |
| 共通テスト制度変更 | 高 | 情報Iの追加など柔軟な科目構造設計（済） |
| 競合の追従 | 中 | ヒートマップUIの特許検討、UX改善の継続投資 |
| サーバーコスト増大 | 中 | Cloudflare Workers/D1で従量課金を最小化 |
| 個人情報漏洩 | 高 | LINE Login活用で自前認証最小化、D1暗号化 |

# PWA Screenshots

PWAマニフェスト用のスクリーンショットを配置するディレクトリ。

## 必要なファイル

| ファイル | サイズ | 説明 |
|---------|--------|------|
| `heatmap-mobile.png` | 390x844 | モバイル版 学習マップ画面 |
| `heatmap-desktop.png` | 1280x720 | デスクトップ版 学習マップ画面 |
| `timekeeper-mobile.png` | 390x844 | モバイル版 タイムキーパー画面 |

## 生成方法

1. `pnpm dev` でローカルサーバーを起動
2. Chrome DevTools → Device Toolbar で各サイズに設定
3. スクリーンショットを撮影して上記ファイル名で保存
